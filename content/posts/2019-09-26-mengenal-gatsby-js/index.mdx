Pembaca mungkin sudah akrab dengan Wordpress, CMS yang sangat populer 
untuk mebuat blog. CMS semacam Wordpress memiliki sistem yang sangat besar. 
Ia memiliki database sendiri, sistem tema sendiri, dan sistem plugin yang 
terpadu. Karena sistem yang besar, terkadang CMS semacam Wordpress terlalu 
"kompleks" untuk membuat website yang sekedar menampilkan halaman HTML. 
Tapi, untuk membuat website yang hanya terdiri dari halaman HTML saja 
secara manual tentu akan merepotkan dan memakan waktu. Disitulah static 
site generator (SSG) muncul sebagai solusi . SSG memiliki sebuah sistem 
dimana kita menulis konten di file yang khusus (biasanya dalam Markdown), 
kemudian ia akan men-generate file-file HTML-nya saja. Dengan SSG, kita 
memiliki web yang ringan (karena hanya terdiri dari file HTML saja), tanpa database.  
  
GatsbyJS merupakan salah satu SSG baru yang menggunakan library ReactJS
