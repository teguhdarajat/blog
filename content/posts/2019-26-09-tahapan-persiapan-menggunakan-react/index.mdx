## Install React
Langkah ini cukup mudah, kita hanya perlu mengetik command dibawah ini:
```
npm install -g create-react-app
```
Setelah selesai buat folder baru beri nama belajar-react dan change directory ke folder 
tersebut lalu klik ketik perintah dibawah ini.
```
create-react-app project-react
```

Setelah itu masuk ke folder project-react. lalu kita coba jalankan apps project-react
kita dengan mengetik command dibawah ini:
```
npm start
```

Selamat sekarang kamu telah mempunyai apps pertama react.
